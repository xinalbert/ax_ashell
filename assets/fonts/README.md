# Bundled Fonts

AxShell embeds a deliberately small set of font files. Font family names below are the internal
names used by GPUI and stored in user configuration.

| Family | Version | Included files | Intended use |
| --- | --- | --- | --- |
| Maple Mono NF CN | bundled release | Regular, Bold | Default terminal font with CJK and Nerd Font coverage |
| Iosevka Term | 8.0.0 | Regular, Bold, Italic, Bold Italic | Compact terminal text and dense English code |
| JetBrains Mono | 2.304 | Regular, Bold, Italic, Bold Italic | General-purpose programming and terminal text |
| Monaspace Neon Var | 1.400 | Variable font | Variable 200-800 weight, 100-125 width, and slant support |

Only the styles selected by AxShell's terminal renderer are bundled. Iosevka Extended, Oblique
and extra weights; JetBrains Mono NL, web and extra weights; and Monaspace Argon, Krypton, Radon
and Xenon are intentionally excluded.

and author files in this directory.
