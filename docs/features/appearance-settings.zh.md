# 外观与设置

## 外观

- 跟随系统主题，或手动选择浅色/深色模式。
- 分别选择浅色主题和深色主题。
- 配置界面与终端字体及字号。
- 选择终端光标和标题栏样式。
- 使用颜色、背景、字体亮度和名称创建自定义主题。

## 终端与工作区偏好

- 右键复制粘贴行为。
- 终端关键词高亮。
- SSH 重试参数和 Shell 偏好。
- 工作区布局锁定和监控面板位置。
- 非激活标签状态色：默认中性色，或让所有标签保留颜色。
- 存在活动传输时的 SFTP 关闭行为。

大多数开关和选择控件会即时保存。仍显示 **Save** 命令的文本表单需要显式保存。

## 关闭设置页

通过标签关闭按钮关闭设置页，或在设置页已打开时再次调用设置快捷键，都会显示确认弹窗。记住的偏好只控制弹窗打开时第二次按设置快捷键的动作：

- 关闭设置页；或
- 保持设置页打开。

记住该动作不会关闭以后出现的确认弹窗。
