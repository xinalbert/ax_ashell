# Appearance And Settings

## Appearance

- Follow the system theme or select light/dark mode manually.
- Choose separate light and dark themes.
- Configure UI and terminal fonts and sizes.
- Select terminal cursor and title-bar styles.
- Create custom themes with color, background, font brightness, and name settings.

## Fonts

AxShell includes a small built-in font set for terminal defaults and Settings font ordering. See [Bundled Fonts](bundled-fonts.md) for source repositories, included styles, and licenses.

## Terminal And Workspace Preferences

- Right-click copy and paste behavior.
- Terminal keyword highlighting.
- SSH retry values and shell preferences.
- Workspace layout locking and monitoring placement.
- Inactive-tab status colors: neutral by default or colored for every tab.
- SFTP close behavior when active transfers exist.

Most switch and selection controls save immediately. Text forms that show a **Save** command still require that explicit action.

## Closing Settings

Closing the Settings tab or invoking the Settings shortcut while Settings is open displays a confirmation dialog every time. The remembered preference controls what a second Settings shortcut does while that dialog is open:

- close Settings, or
- keep Settings open.

The remembered action does not disable future confirmation dialogs.
